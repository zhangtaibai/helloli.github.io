CSS background属性

https://www.w3.org/TR/2014/CR-css3-background-20140909/#backgrounds

一、background-color(背景颜色)

  1. transparent(默认透明)
  2. color_name(red)
  3. hex_number(#0F0)
  4. rgb_number(rgb(0,0,255))
  5. rgba_number(rgba(0,0,255,.5))
  6. hsl_number(hsl(240,100%,50%))
  7. hsla_number(hsla(240,100%,50%,.5))
  8. inherit

二、background-image(指定背景图像)

  1. url(360.png) url(360.png),url(smile.gif)
  2. none(默认值)
  3. inherit

三、background-repeat(重复方式)

  1. repeat(默认值)
  2. repeat-x
  3. repeat-y
  4. no-repeat
  5. inherit

四、background-position(背景图像起始位置)

  1. 25% 50%(默认值0% 0%)
  2. center right
  3. 50px 50px

五、background-attachment(滚动)

  1. scroll(默认值,随着屏幕滚动)
  2. fixed(不随屏幕滚动)
  3. inherit
  4. local

六、background-size(CSS3背景图片尺寸)

  1. length(200px 80px)(默认值auto)
  2. percentage(80% 80%)
  3. cover(覆盖，不留白))
  4. contain(包含，显示全)

七、background-origin(CSS3背景图片定位区域)

  1. border-box
  2. padding-box(默认值)
  3. content-box

八、background-clip(CSS3背景的绘制区域)

  1. border-box(默认值)
  2. padding-box
  3. content-box

九、background(混合写法)

  1. url(360.png) #CCC no-repeat fixed center
